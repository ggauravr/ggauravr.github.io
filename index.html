Projects

  • HashAround: A Twitter and Google-Maps API mashup

    A webapp built in the OSU Hackathon Autumn '13, that gives the latest tweets about(using hashes used in the tweets) and from the place(using geo-location provided by Twitter API), selected on the map.


    Tools/Technology :

    • API : Twitter API v1.1, Google Maps API
    • Frontend : Yeoman, Sass, HTML5 Geolocation
    • Hosting : Google App Engine(used PHP)
    URL : http://www.hash-around.appspot.com

  • K-Nearest Neighbor Classifier for Semeion Digits-Dataset

    Tried out K-Nearest Classifier classifier for digit recognition, trained on the Semeion Dataset, from UCI repos. It's a basic version of knn using 5 neighbors, and the objective was to learn features and usage of pandas DataFrames, numpy arrays, sklearn tools like confusion_matrix for evaluation and pyplot


    Tools/Technology:

    • Modules : numpy, pandas, sklearn, pyplot
    URL : http://github.com/ggauravr/knn-digit-classifier
